# 起步

## TypeScript 是什么?

::list{type="success"}

- 由微软开发的自由和开源的编程语言
- `JavaScript` 的超集，扩展了 `JavaScript` 的语法
- 能为开发者提供了良好的类型提示，帮助提高代码质量，可读性和维护性

::

:Space
